# 뉴스레터 정통망법 준수 체크리스트

> 정보통신망 이용촉진 및 정보보호 등에 관한 법률 (이하 "정통망법")
> 제50조(영리목적의 광고성 정보 전송 제한), 제50조의5, 시행령 제62조

## ✅ 1단계: 수집 (가입 폼)

- [x] **명시적 동의 (opt-in)**: 체크박스 사전 미체크, 사용자가 직접 체크해야 가입 (sec 50①)
- [x] **동의 문구 명시**: 발송 빈도, 발송 내용, 해지 가능 안내 포함
- [x] **동의 증빙 저장**: `subscribers.consent_at`, `consent_text`, `ip_hash`, `user_agent` 4개 필드 저장
- [x] **이메일 형식 검증**: 클라이언트 + 서버 양쪽 (`/^[^\s@]+@[^\s@]+\.[^\s@]+$/`)
- [ ] **만 14세 미만 동의 처리**: 별도 법정대리인 동의 절차 — 노동법 사이트 특성상 만 14세 미만 가입 거의 없음, 정책으로 가입 자체 제한

## ✅ 2단계: 확인 (Double opt-in)

- [x] **DB 컬럼 준비**: `confirm_token`, `confirmed_at`, status enum 준비됨
- [ ] **확인 이메일 발송**: 5/3 이후 sprint — Resend 연동 후 confirm_token 포함 링크 발송
- [ ] **확인 링크 클릭 처리**: `/api/subscribers/confirm?token=...` route 추가 → status='confirmed'로 업데이트 + confirmed_at 기록

## ✅ 3단계: 발송 (광고성 정보 전송)

- [ ] **제목에 (광고) 표기**: 영리 목적 정보면 제목 앞에 `(광고)` (sec 50④)
  - 예: `(광고) 노란봉투법 가이드 주간 인사이트 — 4월 4주차`
  - 노무 사이트의 자체 콘텐츠 알림은 광고로 분류될 수 있음 (보수적으로 표기)
- [ ] **발신자 정보**: 이메일 본문에 노무법인 위너스 명칭, 사업자번호, 주소, 연락처 명시
- [ ] **수신거부 링크**: 모든 발송 메일 하단에 1-click unsubscribe 링크 (`/api/subscribers/unsubscribe?token=...`)
- [ ] **수신거부 즉시 처리**: 클릭 → status='unsubscribed', unsubscribed_at 기록, 향후 발송 차단
- [ ] **야간 발송 제한**: 영리 광고는 21시~익일 8시 추가 동의 없이 발송 금지 (sec 50⑧)
  - 주간 발송 시간을 09:00~21:00 KST로 제한

## ✅ 4단계: 사후 관리

- [ ] **수신동의 정기 확인**: 2년마다 동의 여부 재확인 (sec 50⑦)
- [ ] **bounce 처리**: 5회 이상 bounce 시 자동 status='bounced'
- [ ] **개인정보 보유 기간**: 가입 해지 후 즉시 삭제 또는 별도 분리 보관
- [ ] **개인정보 처리방침 갱신**: 사이트 footer에 뉴스레터 항목 추가 (수집 항목, 이용 목적, 보유 기간)

## ⚠️ 위반 시 처벌

- 정통망법 제50조 위반: 3,000만원 이하 과태료 (시행령 제66조)
- 노무법인 위너스 같은 전문 서비스가 위반 시 신뢰도 타격 (메타-역효과)

## 우선순위

1. **5/1~5/2** — 수집(1단계) 완료 ✅ 진행 중
2. **5/3** — 발송(3단계) 구현, Resend 연동, 확인 이메일 + (광고) 표기 + unsubscribe 링크
3. **5/4** — 개인정보 처리방침 갱신, 야간 발송 제한 cron 룰
