# 조문 매핑 템플릿

취업규칙 비교 검토 시 표준취업규칙과 대상 취업규칙 간 조문 대응 관계를 정리하는 템플릿이다.

## 사용법

1. 대상 취업규칙의 각 조문을 순서대로 나열한다
2. 표준취업규칙의 대응 조문을 매핑한다
3. 대응 상태를 표시한다
4. 표준취업규칙에는 있으나 대상 취업규칙에 없는 조문을 별도 식별한다

## 대응 상태 코드

- ✅ **완전 대응**: 표준취업규칙과 내용이 유사하게 대응
- ⚠️ **부분 대응**: 대응은 되지만 내용의 차이가 있음 (상세 비교 필요)
- ❌ **누락**: 표준취업규칙에는 있으나 대상 취업규칙에 없음
- ➕ **사업장 고유**: 대상 취업규칙에만 있는 조항 (표준취업규칙에 대응 없음)
- 🔄 **조문 통합/분리**: 여러 조문이 하나로 통합되었거나 하나가 분리됨

## 필수/선택 구분

표준취업규칙의 각 조문은 다음과 같이 구분된다:
- **[필수]**: 근로기준법 제93조 각호에 해당하여 필수 기재 필요
- **[선택]**: 필수사항은 아니나 규정하는 것이 권장됨
- **[필수, 선택]**: 일부 내용은 필수, 일부는 선택

누락 조문 중 [필수]인 것은 우선적으로 보완이 필요하다.

|---|---|---|---|---|---|

|---|---|---|---|
| 1 | 제XX조(OOO) | 확인 필요 | |
| 2 | ... | ... | ... |
