# R5 iOS Safari 스모크 체크리스트 (베이스라인 + 회귀)

> **목적**: iOS Safari < 16.4 lookbehind regex 우회 + 모바일 UI 회귀 점검.
> **방법**: 실기 iPhone에서 노란봉투법.com 5흐름 통과 여부 체크.
> **사용 시점**: ① Phase 0 베이스라인 측정 (현재) ② Phase 4 검증 시 회귀 확인

---

## 환경
- 기기: iPhone (모델/iOS 버전 기록)
- 브라우저: Safari (시크릿 창 권장 — 캐시 영향 제거)
- 네트워크: 모바일 데이터 / Wi-Fi 둘 다

| 항목 | 측정값 |
|------|--------|
| iPhone 모델 | (예: iPhone 12) |
| iOS 버전 | (예: 16.7.5) |
| 브라우저 | Safari |
| 측정 일시 | YYYY-MM-DD HH:MM |
| 측정자 | |

---

## 흐름 1 — 홈 → AI 챗봇 → FAQ 인용 클릭
- [ ] 홈(`/`) 정상 로딩 (히어로 이미지, 메뉴 표시)
- [ ] 상단 메뉴 → "AI" 탭 진입
- [ ] 챗봇 입력 "퇴직금 10년" 송신
- [ ] 응답에 인용 마크 `[FAQ#1234]` 등장 (텍스트가 잘리지 않음)
- [ ] 인용 마크 클릭 → 팝오버/페이지 정상 (lookbehind regex 우회 확인)
- [ ] **PASS / FAIL** + 메모:

## 흐름 2 — FAQ 페이지 → 카테고리 → 상세
- [ ] `/faq` 진입, 카테고리 탭 표시
- [ ] "퇴직금" 카테고리 클릭 (한글 URL 디코딩)
- [ ] 카테고리 페이지 정상 로딩
- [ ] 개별 FAQ 클릭 → 상세 표시
- [ ] **PASS / FAIL** + 메모:

## 흐름 3 — 판례검색(`/database`) → 판례 상세
- [ ] `/database` 진입, 검색창 표시
- [ ] 검색어 "부당해고" 입력 → top-5 결과
- [ ] 카드 클릭 → `/decisions/[id]` 진입
- [ ] 핵심쟁점 본문 풀버전 표시 (truncation 없음)
- [ ] 가나다 마커 (가., 나., 다.) 줄바꿈 정상
- [ ] **PASS / FAIL** + 메모:

## 흐름 4 — 노란봉투법 안내 → 챗봇 도구 호출
- [ ] 챗봇에 "노란봉투법 시행일" 입력
- [ ] 응답에 시행일 (2026.X.X) + 인용 마크
- [ ] 챗봇에 "통상임금 250 30 10 계산" 입력
- [ ] 응답에 수치 결과 표시 (계산기 도구 호출됨)
- [ ] **PASS / FAIL** + 메모:

## 흐름 5 — 비교분석(`/sanction`) 진입
- [ ] 챗봇에 "비슷한 판례 보고싶어" 입력
- [ ] 응답에 `/sanction` 링크 등장
- [ ] 링크 클릭 → 비교분석 페이지 진입 (현재는 외부 도메인, Phase 4 후엔 내부 path)
- [ ] 페이지 정상 동작
- [ ] **PASS / FAIL** + 메모:

## 알려진 이슈 / 회피 사항
- iOS Safari < 16.4 — JS lookbehind regex `(?<![A-Za-z])` 미지원 → 캡처 보존 패턴으로 우회 (commit `2a0b75a`)
- Safari 한글 URL 디코딩 시 더블 인코딩 → `/faq/%EC%97%B0%EC%B0%A8...` 정상 표시 확인
- iOS Safari Tap delay 250ms — 클릭 반응 느릴 수 있음 (정상)
